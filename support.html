build troubleshooting

issue possible solution
shows error after switching to logic turn off "check for active click" and "check for active solo" in bounce settings.
logic stopped midway use the "choose track number to start from" function to resume from where it stopped.
fails to get track names or switch verify accessibility permissions. try removing and re-granting bouncrrr's accessibility permission in system preferences.
logic keeps crashing bouncrrr automates logic but can't prevent internal crashes. ensure plugins are stable or try a different logic version.
bounces include sends from other tracks this can happen with certain soloing setups. try muting all tracks first, ensuring necessary folders/groups remain audible for the target stems.
